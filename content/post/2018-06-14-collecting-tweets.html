In this post I am going to download tweets using tweetR package.

  1. You have to have a twitter account. If you do not have, create a twitter account on twitter.com.

  2. Create a twitter application on apps.twitter.com. There, you have to insert a number of information that will be used to collect tweets. Once, you created an application, you will have access to API Key and API Secret in the Keys and Access Tokens tab. In the same tab, you can create access tokens too (no picture is provided here).

  1. Go to RStudio: install and attach tweetR package.
install.packages("twitteR")
library(twitteR)
  1. From step 2, insert API Key, API Secret, Access Token, and Access Secret into the following code chunk.
api_key <- "YOUR_API_KEY"
api_secret <- "YOUR_API_SECRET"
access_token <- "YOUR_ACCESS_TOKEN"
access_secret <- "YOUR_ACCESS_SECRET"
  1. Now, setup your twitter details with twitteR
setup_twitter_oauth(consumer_key = api_key, consumer_secret = api_secret, access_token=access_token, access_secret=access_secret)
  1. Collect tweets of your interestâ€¦ That simple!
tw_afghanistan <- searchTwitteR(searchString = "#Afghanistan", n = 5)

At this stage, you will have the tweets in the form of a list. You can check them out like any other list. For example, look at the first tweet tw_afghanistan[[1]].
