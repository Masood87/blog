Explore Correlation Among Variables with Corrplot

2018/06/14

Exploring relationship among variables in a data set is never easier with corrplot package and function of the same name. corrplot takes in a correlation matrix and returns a visual representation of that correlation matrix.

Try it for yourself.

  1. Install and load corrplot package
install.packages("corrplot")
library(corrplot)
  1. Estimate correlation between all variables of mtcars dataset using cor, and then plot the correlations using corrplot(). You have to specify a method, and there are quite a few of themâ€“here I demonstrate two of them: square and number
mat <- cor(mtcars)
corrplot(mat, method = "square")

corrplot(mat, method = "number")

To explore more, see the help (?corrplot) and visit the Introduction to corrplot Package post on CRAN.
